# Deploy Gitthub-Workflow Skill to Claude.ai

**Purpose:** Install the gitthub-workflow skill in Claude.ai to generate comprehensive, structured workflows directly in the web interface.

**Target Completion Time:** 10-15 minutes
**Total Steps:** 4

---

## Step 1: Verify Account and Enable Code Execution

**Instruction:**

```text
Go to Claude.ai settings and verify you have a Pro, Max, Team, or Enterprise
subscription. Navigate to the Skills section and ensure "Code execution" is
enabled. If not enabled, toggle it on to allow skills to function properly.
```

**Deliverable:** _Claude.ai account verified with valid subscription tier and code execution enabled_

---

## Step 2: Upload Gitthub-Workflow Skill ZIP

**Instruction:**

```text
In the Skills section, click "Upload skill" button and select the gitthub-workflow.zip
file from your computer. Wait for the upload to complete and verify the skill
appears in your skills list with the name "gitthub-workflow".
```

**Deliverable:** _Gitthub-workflow skill successfully uploaded and visible in Claude.ai skills list_

---

## Step 3: Enable the Skill

**Instruction:**

```text
Locate the gitthub-workflow skill in your skills list and toggle it to "enabled"
or "on" status. Confirm the skill shows as active and ready to use in your
Claude conversations.
```

**Deliverable:** _Gitthub-workflow skill enabled and active in your Claude.ai account_

---

## Step 4: Test Workflow Generation

**Instruction:**

```text
Test the skill by asking Claude: "create a gitthub workflow to compare database
options". Verify the skill triggers correctly with discovery questions, generates
an outline for approval, and creates a complete formatted workflow when you
proceed through the process.
```

**Deliverable:** _Successfully generated test workflow confirming skill is working correctly_
